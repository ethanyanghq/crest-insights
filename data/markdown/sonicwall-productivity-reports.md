## Business Challenge

Sonicwall provides a Content Filter Service in the firewalls. SonicWall CFS compares requested web sites against a massive database in the cloud containing millions of rated URLs, IP addresses and websites. It provides administrators with the tools to create and apply policies that allow or deny access to sites based on individual or group identity, or by time of day, for over 50 predefined categories. As all traffic in the firewall is tagged with these 50+ categories it becomes difficult to provide insights on the users/websites/web categories and reports.

## Customer Solution

Crest Data implemented a UI feature of Category groups. All the categories can be grouped into a particular group. Ex. Productive, Unproductive etc. Admin can also create custom groups based on their requirements. Using groups, now network activity can be mapped with a group using categories. Crest data implemented listing pages of Users, websites and web categories and reports page which has the mapping to a CFS group. Following features were implemented:

1. CFS category groups

2. Reports by Connection, Browse time and Data transfer

3. Users, Websites and Web categories listing

4. Session Logs details
